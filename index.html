Core Architecture

Two Surfaces, One Platform

Antigravity's fundamental innovation: separate spaces for hands-on coding and autonomous agent work.

Synchronous Workflow
Editor View

The traditional, hands-on coding interface you already know — but supercharged with AI at every keystroke.

  • Tab autocomplete — state-of-the-art inline suggestions
  • Natural language code commands
  • Context-aware, configurable agent
  • Inline code explanations and refactoring
  • Works like VS Code — all extensions compatible
Asynchronous Workflow
Manager Surface

The real differentiator — a dedicated interface for orchestrating multiple agents working asynchronously across workspaces.

  • Spawn multiple agents simultaneously
  • Observe progress via Artifacts (not raw logs)
  • Comment on artifacts like a Google Doc
  • Agent learns from your project knowledge base
  • Mission control for all running tasks

Features

Cross-Surface Agent Capabilities

Cross-Surface Agents

One agent modifies code, runs terminal commands, and validates in the browser — all without switching context.

Parallel Agent Teams

Manager Surface: spawn 3 agents simultaneously on different tasks. Monitor all from one central view.

Knowledge Base

Agents save useful code snippets, patterns, and decisions to improve future tasks automatically.

Async Execution

Delegate complex tasks and let agents work in the background while you focus on other work.

Trust Architecture

Progressive autonomy controls. Set review policies from manual approval to fully autonomous.
